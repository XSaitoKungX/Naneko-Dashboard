Latest Version [v7.2.0-stable]

Was ist Neu?

In diesem Update führen wir zwei neue Funktionen für unseren Bot ein, das Ticketsystem und das Anti-Raid-System
Das Ticketsystem in Naneko ist vollständig reaktionsbasiert!
Sobald Sie die Einrichtung abgeschlossen haben, können Sie Reaktionen verwenden, um das Ticketsystem zu verwenden
Derzeit enthält das Ticketsystem nur grundlegende Funktionen, aber wir werden in zukünftigen Updates erweiterte Funktionen hinzufügen!
Um die Befehle zur Verwendung dieser neuen Funktionen auszuprobieren, besuchen Sie den Befehlsabschnitt auf dieser Website.

Wichtige Info

Uns sind die im Musiksystem auftretenden Fehler bekannt. Die Fehler treten aufgrund einiger Probleme mit dem Paket auf, das wir für Musik in unserem Bot verwenden. Im Moment haben wir einige temporäre Korrekturen an der Musik vorgenommen, die Korrekturen sind vorübergehend, daher ist die Musik jetzt etwas langsamer als im Vergleich zu früher. Wir warten darauf, dass die Paketentwickler einen Fix veröffentlichen, damit wir das Musiksystem in Naneko reparieren können.
